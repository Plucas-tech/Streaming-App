# 🎬 Streaming APP (React Native)

Este projeto é um clone da interface do Prime Video, desenvolvido como trabalho acadêmico para a disciplina de Programação para Dispositivos Móveis.

## 🎯 Objetivo

O objetivo do projeto é praticar o desenvolvimento de aplicações mobile utilizando React Native, recriando a experiência visual de uma plataforma de streaming.

## 🛠️ Tecnologias utilizadas

* React Native
* Expo
* JavaScript

## 📱 Funcionalidades

* Tela inicial inspirada no Prime Video
* Listagem de filmes e séries
* Navegação entre telas
* Interface moderna e responsiva
* Componentes reutilizáveis

## 📌 Observações

Este projeto foi desenvolvido apenas para fins educacionais e não possui vínculo oficial com o Prime Video.
